# Hilmys's Life

Game ini merupakan game simulasi kehidupan Hilmys.

### Prerequisites

Cara Compile dan Run di Linux atau Mac OS

```
 $gcc addGame.c Hilmys.c Hilmys.h Kegiatan.c main.c -o main
 $./main
```
